# Cybercoin :computer:
A blockchain implementation for the Cybercoin (CYB) cryptocurrency at CB CYBER.

## :pushpin: Installation
```
git clone https://github.com/milesrack/cybercoin
cd cybercoin
python3 setup.py install
```
